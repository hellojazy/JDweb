# 京东补货单生成器操作说明

## 1. 维护商品资料

1. 打开“商品管理”页面。
2. 点击“商品导入模板”，下载空模板。
3. 按表头填写商品资料，至少保证 `商品SKU`、`商品名称`、`外箱箱规(X件/箱)` 正确。
4. 上传商品表并点击“导入并更新商品”。

导入规则：`商品SKU` 是唯一 ID。每次导入都以本次表格为准，表格里有的商品会新增或覆盖，表格里没有的旧商品会从商品库删除。

## 2. 生成补货表

1. 打开“补货生成”页面。
2. 点击“库存导入模板”，可下载只含有效表头的库存模板。
3. 上传京东后台导出的最新库存 Excel。
4. 按需调整生成日期、热销判定 14 日销量、热销品周转周期、普通品周转周期、B 仓周转周期。
5. 点击“生成表格”。
6. 生成完成后，可预览并下载“手工作业单”和“京东入 B 仓”两个 Excel。

## 3. 查看历史

1. 打开“生成历史”页面。
2. 记录每 10 条分页展示。
3. 选择某条历史记录后，可预览当时生成的数据，并重新下载对应 Excel。

## 4. 注意事项

- 库存导入时，配送中心为“全国”的行只用于热销判断和汇总参考，不生成分仓补货。
- B 仓库存只按配送中心“成都补货B”统计。
- 采购需求会扣减采购在途数量。
- B 仓不销售，只作为调拨到分仓的库存池。
- 模板文件只保留表头，不包含真实 SKU、库存、销量、采购价等关键数据。
